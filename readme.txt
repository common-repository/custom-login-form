Customize the WordPress Login Form.

== Description ==
Custom Login Form has the following features:

* Change the Login Logo
* Styling your Login Button
* Change the Background Color
* Add your Copyright to the Footer

== Installation ==
1. Upload the plugin folder "custom-login-form" to the /wp-content/plugins/ directory.
2. Activate the plugin through the "Plugins -> Installed plugins -> Custom Login Form" menu in WordPress.
3. Find "Custom Login Form" Plugin in -> Settings options.
